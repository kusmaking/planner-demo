# v18.62k project workbench stable floating window

Denne patchen er bygget på v18.62e, ikke på de ustabile v18.62f-j-forsøkene.

Endringer:
- prosjektvinduet er et fast, flyttbart vindu innenfor skjermen
- tydelig lukkeknapp øverst til høyre og nederst
- kan dras fritt horisontalt og vertikalt fra toppfeltet
- kan endres i størrelse fra kanter/hjørner
- fullvisning skjer kun via egen knapp
- nullstill setter vinduet tilbake til trygg standardplassering
- ingen endring i tildelingslogikk, overbooking, Supabase, RLS eller kalenderdata
